m0n0wall Documentation Project Wish List

This list outlines the current priorities of the m0n0wall Documentation Project.  If you feel you can contribute in any of these areas, please email Chris Buechler at m0n0wall@chrisbuechler.com.

1.  Fill in blank sections of Users Guide.
2.  Instructions on configuring m0n0wall IPsec VPN with third party devices.  m0n0wall will interoperate with any IPsec VPN device.  I want to see more documentation of how to configure both ends.  A section of the wiki has been devoted to this purpose.  See an example of what I am looking for.
3.  Example configurations, step by step. 

I have already (mostly) completed the Quick Start Guide, and am working on filling up the Example Configurations chapter with numerous helpful examples of common configurations.

I would like to see step by step instructions on a variety of common configurations, as well as any odd ball configurations people may be using. 

This includes, but is not limited to:

  • Configuring wireless on an existing m0n0wall setup
  • Building to building bridge
  • Multiple LAN interfaces

4.  Troubleshooting - I want to get together a troubleshooting section listing common problems and steps to determine what the problem is and how to solve it. 

Chris Buechler
